# Playst's babel-preset

> A Playst's babel preset for transforming JavaScript.

## Install

```sh
$ npm install --save-dev @playst/babel-preset
```

## Usage

### By using `.babelrc`

**.babelrc**

```json
{
  "presets": ["@playst/babel-preset"]
}
```

### By using CLI

```sh
$ babel script.js --presets @playst/babel-preset
```
